# Data Centers and Superfund Sites

# Data

The data draws on the following primary sources:
1. Data Centers 
Source: [Google Sheet — Compiled data center inventory](https://docs.google.com/spreadsheets/d/1JJ6kcVo-NjlAYtznwHOki2DVl4WWV6lhy-eXhFCdKKU/edit?pli=1&gid=386766486#gid=386766486)
A crowdsourced and aggregated inventory of data center location across the United States.

2. EPA Superfund Sites (National Priority List)
Source: [EPA Superfund Redevelopment Mapper — ArcGIS Feature Service](https://www.arcgis.com/home/item.html?id=d6e1591d9a424f1fa6d95a02095a06d7)
The EPA's National Priorities List (NLP) is the authoritiative register of hazardous waste sites in teh United States that are candidates for, or currently undergoing, federally funded cleanup.

3. FracTracker
Source: [FracTracker Alliance](https://fractracker.org/)
FracTracker data is used as a supplemental layer to map industrial
infrastructure and chemical-adjacent activity in proximity to both data center
locations and Superfund sites.
